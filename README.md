# Аналитическая система взаимозависимости ставок заимствования в РФ и РК

В данном репозитории находится код для построения дашборда, способного помочь при оценке рисков, а так же ноутбуки с выявлением взаимозависимостей.

### Инструкция по использованию

1. Установите необходимые библиотеки, если они еще не установлены:
```bash
pip install -r requirements.txt
```

2. Установите веса ARIMA по ссылке из файла arima_weights.

3. Чтобы запустить программу, в терминале выполните:
```bash
cd Метрики
streamlit run dashboard.py
```
